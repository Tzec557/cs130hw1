In this project, you will implement triangle rasterization with a z-buffer and
perspective correct color interpolation.  You should implement this in the file
raster.cpp.  You may add helper functions in this file as needed.  

IMPORTANT NOTE: You should *not* modify any files other than "raster.cpp".  We
will be compiling your "raster.cpp" file against our code base. If you modify
other files for your solution, it may fail to compile or work properly in our
code base.

* Implementation

For this project, you should implement the following features:
- Triangle rasterization.
- Depth testing with a z-buffer.
- Perspective-correct interpolation of colors.

* Compilation

Run scons to compile the project:
%> scons

* Running and Automatic Grading

In this project, there is only one executable which serves as the grading script
as well.  You can run it as:
%> ./raster
